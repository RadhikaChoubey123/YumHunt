YumHunt is a recipe search web application that allows users to search for dishes and view complete details such as ingredients, instructions, and images. The app is built using React.js, styled with Tailwind CSS, and optimized with React Query for fast and efficient data fetching. It delivers a smooth, responsive, and user-friendly cooking experience.
